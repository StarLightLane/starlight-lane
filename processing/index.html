How we price

Interchange-plus, nothing more.

Every credit card transaction has three costs. Interchange (what the card networks charge), assessments (what the networks add on top), and the processor markup (the negotiable part). We show you all three, every month, on every statement. The only variable between processors is that last slice — and ours starts at 0.15%.

Same interchange cost as everyone else — set by Visa/MC/Amex, not negotiable with any processor
Same assessments — roughly 0.14%, also fixed by the card networks
Our markup starts at 0.15% — that’s the only number you’re actually comparing between processors
Nothing is hidden — every statement shows interchange, assessments, and our markup separately
